GroovyBot

GroovyBot is a Google Wave Robot that supports evaluation of Groovy scripts in the wave. GroovyBot can be added to a wave as a participant. Its address is groovybot@appspot.com. Currently, the following features are supported:

ScriptBlips
Create a blip starting with the prefix "!groovy" followed by actual groovy code. The code will be executed and the result will be displayed in the wave. Example:
!groovy
def x = 100
def y = 500
println x/y
TemplateBlips
Create a blip starting with the prefix "!gtemplate" if you want the blip to be evaluated as a Groovy Template. The result will be displayed in the wave. Example:
!gtemplate
<%= 
    def cal = Calendar.instance
    def hour = cal.get(Calendar.HOUR_OF_DAY)
    def greet = hour < 12 ? 'Good morning' : 'Hello' 
    out.print greet
%> World
GroovyBot Gadget
Create a blip starting with the prefix "!ggadget" to include a nice gadget, that lets you execute groovy code.
